# react-router-form
React multipage form using React Router.


### Demo

https://react-router-multipage-form.herokuapp.com/

### Author

Monika Brzostek


### Installation

```bash

git clone https://github.com/MonikaBrzostek/react-router-form.git
cd react-router-form
npm install
```

To run in development mode:-

```bash
npm start
```
